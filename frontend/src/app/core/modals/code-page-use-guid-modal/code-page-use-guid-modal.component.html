User Manual

Description

This window allows you to execute code in multiple programming languages (JavaScript, Python, C++, PHP) through your web browser. You can also use this platform to perform file transformations.

To ensure your code works correctly, please follow the rules below:

Rules

1. You are limited to 5 files as input and output.

2. Use the global variables INPUT_FILE_PATH_(index) and OUTPUT_FILE_PATH_(index) to access the selected file paths and generate output files.

3. Remember, global variable indices start at 0.

4. Select a file format to generate your output file correctly.

5. The order of input and output files during selection must match the order of global variable indices in your code.

Example

def generate_yml():
    json_file = Path(INPUT_FILE_PATH_0)    # first selected file must be the json file
    png_file = Path(INPUT_FILE_PATH_2)     # the png file must be the third one selected
    yml_file = Path(OUTPUT_FILE_PATH_0)    # must be the first selected output format
